# Resource Groups Tagging API

**Protocol:** JSON 1.1 (`X-Amz-Target: ResourceGroupsTaggingAPI_*`)
**Endpoint:** `POST http://localhost:4566/`

The Resource Groups Tagging API provides a unified interface to query, add, and remove tags across resources from many AWS services.

## Supported Actions

<!-- floci:actions:start -->
| Action |
| --- |
| `GetResources` |
| `TagResources` |
| `UntagResources` |
| `GetTagKeys` |
| `GetTagValues` |
